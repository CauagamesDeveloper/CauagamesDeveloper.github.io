Notey, o editor de texto simples

Notey é um editor de texto simples que escreve e edita textos.. Nossa mas isso foi bem anticlimático...
Ele foi feito para os sistemas Windows Vista e Windows 7, se conseguir rodar isso em outro Windows, provavelmente vai ver alguns bugs.

Ele ainda não está disponível para download pelo fato de haver bugs na UI.

Shattered Reality Studios (SRS)

Shattered Reality Studios é uma equipe feita por mim e pelo Herocrife para fazermos jogos com qualidade de ponta (literalmente) enquanto os jogos sejam acessíveis para todos. Criado em 2021, ideias de ótimos jogos, mas sem o hardware necessário para a construção deles.

Hech Simulator

Hech Simulator é um jogo produzido pela Shattered Reality Studios (SRS) feito inicialmente no Roblox, mas havendo seu desenvolvimento interrompido devido a falta de código. O Hech Simulator está atualmente arquivado para o próximo desenvolvimento, agora usando uma tecnologia voxel usando o RPG In A Box.

O jogo tinha previsão de lançamento no final de 2022.
